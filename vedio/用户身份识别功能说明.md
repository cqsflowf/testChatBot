# 用户身份识别功能说明

## 功能概述

本功能实现了基于用户语言特征的成人/儿童身份识别，并根据识别结果自动调整回复风格，使语音助手能够更好地适应不同年龄段的用户。

## 核心模块

### 1. 用户身份识别器 (UserIdentityRecognizer)

**文件位置**: `src/voice_dialog/modules/user_profile.py`

**主要功能**:
- 基于关键词匹配识别用户身份
- 基于句式模式识别用户身份
- 支持持续学习和用户画像更新
- 提供识别置信度评估

**识别特征**:

#### 儿童特征
- **称呼类**: 妈妈、爸爸、爷爷、奶奶、老师等
- **童趣表达**: 呀、呢、啦、哇、好棒、好开心等
- **儿童话题**: 玩具、动画片、幼儿园、作业等
- **句式特征**: 短句、疑问句、征求意见等

#### 成人特征
- **工作相关**: 公司、会议、项目、报告、领导等
- **生活相关**: 房贷、理财、教育、健康等
- **成人表达**: 压力、焦虑、考虑、分析等
- **专业词汇**: 系统、政策、市场、行业等

### 2. 用户画像 (UserProfile)

**数据结构**:
```python
class UserProfile:
    user_type: UserType          # 用户类型 (ADULT/CHILD/UNKNOWN)
    confidence: float            # 识别置信度 (0.0-1.0)
    evidence: List[str]          # 识别依据
    history: List[Dict]          # 历史对话记录
```

### 3. LLM集成

**修改文件**: `src/voice_dialog/modules/llm_planner.py`

**集成点**:
- `plan()` 方法: 非流式任务规划
- `plan_stream()` 方法: 流式任务规划

**工作流程**:
1. 接收用户输入文本
2. 调用 `UserIdentityRecognizer.recognize()` 识别用户身份
3. 构建用户身份上下文提示
4. 将上下文注入到 System Prompt
5. LLM 根据用户身份生成差异化回复

## System Prompt 增强

### 儿童用户提示
```
【用户身份】
识别到用户可能是儿童（置信度：XX%）。
请用儿童的口吻回答：
- 语气亲切温柔，像大姐姐/大哥哥一样
- 用词简单易懂，避免复杂词汇
- 多用比喻和生动的例子
- 可以用"呀"、"呢"、"啦"等语气词
- 回答要有趣味性，可以适当加入童趣元素
- 对小朋友要有耐心和鼓励
```

### 成人用户提示
```
【用户身份】
识别到用户可能是成人（置信度：XX%）。
请用成人的口吻回答：
- 语气专业理性，条理清晰
- 直接高效，不啰嗦
- 可以使用专业术语
- 提供实用的建议和解决方案
```

## 使用示例

### 示例1: 儿童用户
**用户输入**: "妈妈，我今天好开心呀！"
**识别结果**: child, 置信度: 1.00
**系统回复**: "哇，听起来好开心呢！今天发生了什么有趣的事情呀？"

### 示例2: 成人用户
**用户输入**: "今天公司开会，项目进度有点慢"
**识别结果**: adult, 置信度: 1.00
**系统回复**: "项目进度延迟确实让人头疼。建议梳理一下关键节点，看看哪些环节可以优化。"

## 技术特点

1. **多维度识别**: 结合关键词、句式、句子长度等多个维度
2. **置信度评估**: 提供识别置信度，支持模糊判断
3. **持续学习**: 记录历史对话，支持用户画像持续优化
4. **无缝集成**: 与现有LLM规划器无缝集成，不影响原有功能
5. **可扩展性**: 易于添加新的识别特征和规则

## 配置说明

无需额外配置，功能默认启用。如需调整识别规则，可修改 `user_profile.py` 中的关键词列表和评分权重。
